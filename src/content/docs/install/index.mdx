Get Orchard running on your machine of choice, then point it at the services you operate.
Orchard *manages* a mint rather than replacing one, so this section is two steps:
**install** the app, then **configure** it to reach each piece — whether you just
built the stack with [New Mint](/new-mint/) or already run your own Bitcoin,
Lightning, or Cashu mint.
